An ordered HTML list

  1. The < a > tag defines a hyperlink, which is used to link from one page to another
  2. The < button > tag defines a clickable button.
  3. The < br > tag inserts a single line break
  4. The < strong > tag Defines important text
  5. The < sup > tag Defines superscripted text
  6. The < table > tag Defines a table
  7. The < video > tag Defines embedded video content
  8. The < p > tag Defines a paragraph
  9. The < del > tag Defines strikethrough text
  10. The < audio > tag Defines sound content
